Name Entry Event Command by MarkyJoe1990

How to use

1. Put the "Name-Entry-Event-Command" folder in your SRPG Studio plugin folder.

2. Put the "keys" folder inside your project's material folder

3. In your events, create an "Execute Script" event command.

4. In the "Script" tab, click "Call Event Command" and in the object name field, type "Name Entry".

5. In the "Original Data" tab, set the unit field to whatever unit whose name you wish to change.

6. OPTIONAL. In the "Property" field of the "Script" tab, create open and closed curly braces "{}", and put whatever arguments specified by each event command in between the braces. Example:

{
	title: "Name Entry",
	keys: "ABCDEFGHIJKLMNOPQRSTUVWXYZ ",
	selfSwitch: "A",
	lengthLimit: 20,
	defaultName: "James",
	id: 1
}

Event Commands

Name Description Original Data Properties
Name Entry Enter a string to be used in other event commands or scripts. Unit title, keys, selfSwitch, lengthLimit, defaultName, id

Properties

title : "String"

String. Optional. The title of the name entry window. Examples would include "Enter your name" and "What's their name?". The default name is "Name Entry"

keys: "Keyboard Buttons"

String. Optional. Specifies what letters, numbers and symbols will be available on the name entry keyboard. For example, "ABCDEF" would have a keyboard lay out of only A, B, C, D, E and F. The default is "ABCDEFGHIJKLMNOPQRSTUVWXYZ "

selfSwitch: "Letter Or Number"

String or Number. Optional. Determines what self switch will be set to true if the player cancels the name entry event. If you don't set this, the player will not be allowed to cancel the event.

lengthLimit : "number"

Number. Optional. Sets the maximum number of characters the player can type in. Default is 20.

defaultName : "String"

String. Optional. Sets the name to use when the name entry pops up. Default is the unit's name.

id : "number"

Number. Optional. Unit's id in the event that they are not available on the current map.
