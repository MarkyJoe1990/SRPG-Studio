Name Entry Event Commands by MarkyJoe1990

What is this?

It's a custom event command that opens up a window and prompts the player to put in a new name for the specified unit.

How do I use it?

Refer to $commandreference.html for instructions on using the even command.

Who do I thank for this?

Aside from me, you can thank CardCaph (Discord: CardCaph#0588) for some of the lovely key button graphics!

What are the controls?

Assuming you are using the default keyboard settings for SRPG Studio: Z selects a key. X back spaces unless the string is empty, in which it cancels the name entry if that's enabled. A and S both toggle upper and lower case. C adds a space unless the keyboard doesn't have a space key. Lastly, the space key confirms the name entry.

Are they any settings I can mess with?

There is a file called "name-entry-settings.js" located inside the "Name-Entry-Event-Command" folder. You can set the font color, which is in hexidecimal. You can also set which set of key graphics are used. KEYS_GRAPHICS_INDEX tells the plugin what row of graphics to use in the keys material folder. You can add more options by simply editing the image.
